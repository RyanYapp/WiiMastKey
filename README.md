# WiiMastKey

Wii parental control reset tool

1. Make sure the Wii's date matches your PC's date
2. On the Wii, open up confirmation number screen

<img width="413" height="254" alt="3" src="https://github.com/user-attachments/assets/188e01e9-52e7-41d7-aea5-1b6e386c386b" />

3. Enter the confirmation number into WiiMastKey & generate the master key

<img width="146" height="189" alt="1.2 Windows" src="https://github.com/user-attachments/assets/3e3e9f73-61ca-4c50-bf73-64ebee4783f7" />

4. On the Wii, enter the master key to disable parental controls
